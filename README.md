# 📊 Bank Transaction Fraud Detection Dashboard (Power BI)

## 🚀 Project Overview

This project is an interactive **Bank Transaction Analysis Dashboard** built using Microsoft Power BI to monitor, analyze, and detect fraudulent transactions.

The dashboard provides deep insights into transaction behavior, fraud patterns, network usage, and performance metrics using real-time-like data visualization.

---

## 🎯 Objective

* Analyze large-scale transaction data
* Identify fraud patterns using key indicators
* Monitor transaction success/failure rates
* Provide business insights for decision-making

---

## 📂 Dataset Details

The dataset includes the following key fields:

* Sender Account ID
* Receiver Account ID
* Transaction Type (Transfer, Withdrawal, Deposit)
* Transaction Amount
* Fraud Flag (True/False)
* Transaction Status (Success/Failed)
* Bandwidth Group (Network performance)
* Network Slice ID
* Time

---

## 📊 Dashboard Features

### 🔹 1. Transaction Overview

* Total Transactions
* Total Amount Processed
* Success vs Failed Transactions

### 🔹 2. Fraud Detection Analysis

* Fraud vs Non-Fraud Transactions
* Conditional formatting (Red = Fraud, Green = Safe)
* Fraud distribution by transaction type

### 🔹 3. Network Performance Insights

* Bandwidth Group Analysis (50–100 Mbps, etc.)
* Network Slice usage
* Performance impact on transaction success

### 🔹 4. Interactive Filters (Slicers)

* Transaction Type
* Transaction Status
* Device Used
* Network Slice ID

👉 Enables dynamic filtering and drill-down analysis
